# studentvue-reporter
 
A tool to help programmatically generate reports for students using the (StudentVue API)[https://github.com/StudentVue/StudentVue.py].

A `credentials.json` file is required. Format:
```json
{
    "<student_name>": {
            "username": "<username>",
            "password": "<password>",
            "domain": "<domain>"
    }
}
```
